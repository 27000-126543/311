## 1. 产品概述

大型公共资源智慧招投标与信用监管平台是一个面向政府公共资源交易领域的全流程数字化招投标系统。系统涵盖招标发布、投标校验、专家抽取、开标评标、中标公示、合同签署、信用评价等全生命周期管理，通过智能化手段提升招投标效率与透明度，保障公共资源交易的公平公正。

- 目标用户：招标人（政府部门/事业单位）、投标人（企业）、评标专家、交易中心管理员、监管部门
- 核心价值：全流程电子化、智能校验防违规、信用监管控风险、数据驱动决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 招标人 | 机构认证注册 | 发布项目、查看自己项目、处理异议、签署合同 |
| 投标人 | 企业认证注册 | 查看公告、投标、查看自己投标和信用分 |
| 评标专家 | 资格审核入库 | 查看分配评标任务、在线打分 |
| 交易中心管理员 | 内部授权 | 全局管理项目发布、专家抽取、投诉处理 |
| 监管部门 | 内部授权 | 查阅所有数据、调整评分规则和信用阈值 |

### 2.2 功能模块

1. **首页大屏**：实时交易数据、成交金额、流标率、专家使用率、信用分分布热力图
2. **招标管理**：项目发布、模板推荐、招标公告、公告推送
3. **投标管理**：资质上传、自动校验、退回机制
4. **专家抽取**：按专业抽取、回避规则、加密存储、定时解密
5. **开标管理**：在线解密、报价展示、过程录像
6. **评标管理**：在线打分、权重汇总、自动排名、评标报告
7. **中标公示**：候选人标记、在线异议、三级审批
8. **合同管理**：在线签署、保证金关联、变更审批
9. **信用评价**：信用分计算、资格限制、预警推送
10. **数据分析**：组合筛选、报表导出

## 3. 核心流程

### 招投标全流程

招标人发布项目 → 系统推荐模板并生成公告 → 投标人提交投标 → 系统自动校验 → 专家抽取（加密） → 开标（解密+展示） → 评标打分 → 自动排名 → 中标公示 → 异议处理 → 合同签署 → 履约监管 → 信用评价

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝色（#0F2B46）+ 金色（#C8A45C）点缀，传递政府平台的权威与公信力
- 辅助色：浅蓝（#E8F0FE）背景、深灰（#1A1A2E）数据面板、绿色（#00C48C）成功状态、红色（#FF4757）警告状态
- 按钮：圆角8px，主按钮深蓝底金边，次按钮透明底蓝边
- 字体：标题使用 Noto Serif SC（宋体风格，庄重感），正文使用 Noto Sans SC（清晰易读）
- 布局：左侧导航 + 顶部状态栏 + 主内容区，数据大屏采用深色背景
- 图标：线性图标，统一1.5px描边，Lucide图标库
- 数据可视化：ECharts图表，深色背景科技感，数据卡片毛玻璃效果

### 4.3 响应式设计

- 桌面端优先，最小宽度1280px
- 大屏模式（1920px+）优化数据大屏展示
- 平板端（768px-1280px）侧边栏折叠为图标模式
- 移动端暂不支持（政府办公场景以PC为主）
